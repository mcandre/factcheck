factcheck - A Factor port of QuickCheck unit test framework

HOMEPAGE

[http://www.yellosoft.us/quickcheck](http://www.yellosoft.us/quickcheck)

REQUIREMENTS

 - Factor 0.94
 - `$FACTOR_ROOT` is set

INSTALLATION

	$ cp -r factcheck $FACTOR_ROOT/extra

EXAMPLE

	$ factor example.factor 
	+++ OK, passed 100 tests.
	*** Failed!
	{ 979866105 }
	+++ OK, passed 100 tests.
